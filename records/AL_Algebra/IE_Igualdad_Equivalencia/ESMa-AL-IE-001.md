# 2. Descripción de la ESMa

## Patrón de razonamiento observado

Algunos estudiantes desarrollan una comprensión operacional del signo igual, interpretándolo principalmente como una señal que indica que deben realizar una operación y proporcionar una respuesta final.

En esta interpretación, el signo igual funciona como un separador entre una expresión matemática y su resultado, en lugar de comprenderse como un símbolo relacional que establece que dos expresiones representan el mismo valor matemático.

Esta concepción puede ser funcional para resolver operaciones aritméticas simples donde el resultado aparece después del signo igual, pero se convierte en una limitante cuando el estudiante enfrenta expresiones abiertas, ecuaciones o situaciones algebraicas donde ambos lados del signo igual contienen información matemática relevante.


## Manifestación observable

Ejemplo de producción matemática escrita por un estudiante de primer semestre de bachillerato:

Expresión:

5/6 - 1/3 + 1/2 

El estudiante inicia el procedimiento:

5/6 - 1/3 =

y obtiene:

5/6 - 1/3 = (5 x 2)/6 =

Posteriormente continúa el procedimiento sin conservar la estructura completa de la expresión original, evidenciando una interpretación fragmentada del proceso matemático:

5/6 - 1/3 = (5 x 2)/6 = 10/6 + 1/2 = (10 x 3)/6 = 30/6


## Pensamiento matemático esperado

La expresión completa debe conservarse como una unidad matemática hasta finalizar todas las operaciones correspondientes.

5/6 - 1/3 + 1/2 

Utilizando denominador común:

5/6 - 1/3 + 1/2 = 5/6 - 2/6 + 3/6 = 6/6 = 1

El signo igual representa equivalencia entre expresiones, no únicamente una instrucción para obtener una respuesta.

# 4. Análisis cognitivo

## Pensamiento observable

El estudiante parece tratar cada transformación intermedia como una conclusión parcial del problema, en lugar de mantener la expresión completa como una relación matemática continua.


## Posible modelo mental

El estudiante puede haber construido la regla implícita:

> Después del signo igual debe aparecer una respuesta.

Este modelo mental surge frecuentemente de experiencias iniciales donde ejercicios como:

3+4 = [ ]

refuerzan la asociación entre el símbolo igual y la obtención de un resultado final.


## Diferencia entre pensamiento esperado y pensamiento observado

| Pensamiento matemático esperado | Pensamiento observado |
|---|---|
| El signo igual representa una relación de equivalencia entre dos expresiones. | El signo igual funciona como indicador de que se obtuvo una respuesta y puede iniciarse una nueva acción matemática. |

# 5. Origen y conocimientos previos relacionados

## Posible origen cognitivo

Durante los primeros años de aprendizaje matemático, muchos estudiantes trabajan principalmente con expresiones del tipo:

a+b=c

donde el lado derecho representa una respuesta.

Aunque esta interpretación funciona en contextos iniciales, puede convertirse posteriormente en un obstáculo para comprender expresiones equivalentes, ecuaciones y estructuras algebraicas.

## Ideas previas relacionadas

- Comprensión del signo igual como operador de cálculo.
- Resolución mecánica de operaciones.
- Poca exposición a expresiones equivalentes.
- Limitada experiencia con ecuaciones tempranas como relaciones entre cantidades.

# 7. Intervención educativa

## Estrategias didácticas sugeridas

- Presentar expresiones donde el signo igual aparezca entre dos expresiones equivalentes.
- Trabajar ejercicios abiertos como:

8+4= [ ]

- Utilizar representaciones visuales de equilibrio e igualdad.
- Promover explicaciones del razonamiento, no únicamente procedimientos.


## Preguntas docentes recomendadas

- ¿Qué significa realmente el signo igual en esta expresión?
- ¿Lo que está antes y después del signo igual representa el mismo valor?
- ¿Podríamos escribir otra expresión con el mismo valor?
- ¿El signo igual siempre significa "aquí viene la respuesta"?


## Representaciones o recursos útiles

- Modelos de balanza matemática.
- Tarjetas de expresiones equivalentes.
- Representaciones gráficas.
- Narrativas educativas.
