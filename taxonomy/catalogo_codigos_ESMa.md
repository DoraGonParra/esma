## Referencia oficial para la clasificación de Errores Sistemáticos en Matemáticas

------------------------------------------------------------------------

## Propósito del catálogo

Este documento funciona como catálogo maestro de consulta para la
creación de nuevos registros ESMa.

Antes de documentar un nuevo Error Sistemático en Matemáticas, se debe
consultar este archivo para utilizar códigos existentes y mantener la
consistencia de la taxonomía.

Los códigos ESMa son identificadores únicos que conservan su identidad
independientemente del idioma del repositorio.

Formato de identificación:

    ESMa-[Área]-[Tema]-[Número]

Ejemplo:

    ESMa-AR-JO-001

Donde:

-   **ESMa** = Error Sistemático en Matemáticas.
-   **AR** = Área matemática.
-   **JO** = Tema específico.
-   **001** = Número consecutivo del registro.

# Reglas para nuevos códigos

1.  No se deben crear nuevos códigos de manera aislada sin actualizar
    previamente la taxonomía oficial.

2.  Si un nuevo concepto matemático no tiene una categoría adecuada,
    debe evaluarse si:

    -   pertenece a un tema existente;
    -   requiere un nuevo tema dentro de un área;
    -   requiere una nueva área matemática.

3.  Los códigos ya existentes deben conservar su significado original
    para mantener la trazabilidad histórica de los registros.
