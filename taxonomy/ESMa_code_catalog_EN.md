## Purpose of the catalog

This document serves as the master reference catalog for creating new
ESMa records.

Before documenting a new Error Sistemático en Matemáticas, this file
should be consulted to use existing codes and maintain consistency
throughout the taxonomy.

ESMa codes are unique identifiers that preserve their identity
regardless of the repository language.

Identification format:

    ESMa-[Area]-[Topic]-[Number]

Example:

    ESMa-AR-JO-001

Where:

-   **ESMa** = Error Sistemático en Matemáticas (original Spanish
    identity of the initiative).
-   **AR** = Mathematical area.
-   **JO** = Specific topic.
-   **001** = Sequential record number.

# Rules for new codes

1.  New codes should not be created independently without first updating
    the official taxonomy.

2.  If a new mathematical concept does not fit an existing category, it
    should be evaluated to determine whether it:

    -   belongs to an existing topic;
    -   requires a new topic within an existing area;
    -   requires the creation of a new mathematical area.

3.  Existing codes must preserve their original meaning to maintain
    historical traceability of documented records.
